Linux commands


The Linux terminal may seem complicated at first, especially for someone used to the graphical interface of Windows, which consists of icons. Although Linux OS also has a graphical interface similar to Windows (with icons too), in the process of ethical hacking we will focus more on the terminal. The Linux terminal is similar to PowerShell or CMD on Windows, and someone who has used these two applications before may find several similarities between them. So, to use Linux, you need to know certain commands.


1. whoami

As the name suggests ("who am I"), this command prints the username of the current user. For example, running the "whoami" command on Android Linux Userland app, it will print "userland", while on a PC, it will display your username (the one you set up when you configured the system).

andrew@DESKTOP-VVG27SC:~$ whoami
andrew


2. pwd

The "pwd" command (print working directory) displays the path of the folder you are currently in.

Just like on Windows, Linux operates on a system of folders and subfolders. Taking Windows as an initial example, the home directory (the one you access when you open your PC) is Desktop, which is essentially a folder. Running the "pwd" command in Windows CMD in the Desktop folder would produce an output like: "C:\Users\John\Desktop". Similarly on Linux, executing "pwd" in the home directory would return something like: "/home/john" or "/home/userland" (for Userland on Android).

If you have another folder inside your home directory, for example, "notes", then running "pwd" while inside that folder will return: "/home/john/notes".

andrew@DESKTOP-VVG27SC:~$ pwd
/home/andrew


3. ls

The ls command (list) displays the contents of the current folder, showing the files in white and the subfolders in blue.

There are different variations of this command, such as: ls -a (list all) – Shows all files, including hidden ones or ls -l (long listing format) – Displays files along with their properties, such as creation date, permissions, size etc.


4. mkdir

The mkdir command (make directory) will create a directory in the current location. So, for exemple, running mkdir notes will create a directory named notes in the folder you are in.


5. cd

The cd (change directory) command allows the user to navigate into a specific folder. Just like on Windows, where you double click a folder to open it, on Linux you use the cd command you access it. For exemple, to enter the folder "notes", you run cd notes. Once inside the folder, you run ls to display its content (the content does not automatically appear when you enter a directory).
There are more versions of this command. The simple cd command (without specifying a folder name) will take you to the home directory (/home/username). Also, the cd .. command will move you one folder back. For exemple, if you are in /home/john/notes folder and enter cd .., you will go to /home/john. Entering again the same command, you will enter /home directory.


6. cat

The cat command will display the contents of a file. For example, if we have a file named "informations.txt", the command cat informations.txt will show its content in the terminal.


7. nano

Nano is a text editing tool. While the cat command displays the content of a file in the terminal (without allowing editing), nano allows us to create and edit files.

First, we need to check if nano is installed by typing nano in the terminal. If it’s not installed, we can install it using sudo apt install nano. Once installed, we can open nano (enter nano in the terminal) to create a new file, enter text, and close it by pressing CTRL+X, then Y to save, and finally entering the file name and pressing ENTER.
To edit an existing file, we type nano followed by the file name: nano informations.txt. After making changes, we close with CTRL+X, confirm saving with Y, and press ENTER.


8. rm

The rm command (remove) deletes a file. For example, the command rm informations.txt will delete the file informations.txt.
Warning: The rm command only works for deleting files, not folders!


9. rmdir

The rmdir (remove directory) command deletes an empty folder. For example, rmdir notes will delete the notes folder, but only if it is empty. To delete a folder that contains files, use rm -r folder_name.


10. cp

The cp command (copy) copies a file from one location to another. For example, if we have the file "todolist.txt" in the /home/john/notes folder and want to copy it to /home/john/important, we use the command cp location_from location_to. In our case:

cp /home/john/notes/todolist.txt /home/john/important

This will create a copy of todolist.txt in the important folder while keeping the original file in the notes folder.

IMPORTANT: When working with file paths, it’s important to know that if the file is in the directory you are currently in, you can simply use its name without specifying the full path (/home/user/...).

Also, if the file is in a nearby directory, you can use .. (one directory back) to simplify the path.

For example, if we are in the "important" directory, our previous command could be written as: cp todolist.txt ../important

Also, we can change the name of the copied file: cp todolist.txt ../important/renamed.txt. This command you copy the file with a different name.


11. mv

The mv command move the file from a place to another. The usage is similar to the cp command: mv location_from location_to.

12. ifconfig

The ifconfig command displays network information, including the IP address. It is the Linux equivalent of ipconfig on Windows. If the command doesn’t work, you may need to install net-tools using: sudo apt install net-tools.


13. wget

The wget command download a file from internet. Usage exemple: wget https://mywebsite.com/file.txt


14. sudo

With sudo command (super user do) you run a command as root. It is not used alone, but in combination with other commands(for exemple sudo apt install hashcat will install hashcat tool)


Other commands

CTRL+C stops the running program. In Windows Subsystem for Linux (WSL), CTRL+C on a selected text will copy the text and right-click will paste it.
